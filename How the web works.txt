* What is HTTP?Hyper Text Transfer Protocol What is a URL?Uniform Resource Locator:  an address for an internet resource.* What is DNS?Domain Name System, it translates URL in IP-adresses* What is a query string?It pasess key-value pairs into a URL* List two HTTP Verbs and their use cases.o GET - get some data from the server o POST - send some data to the server * What is an HTTP request?It is a request from a client to a server based on the http protocol* What is an HTTP response?It the response from the server to the client based on the http protocol* What is an HTTP header? Give a couple examples of request and response headers you have seen.Headers provide additional information about the request or the responseo Request headers: Host, User-Agent, Cookie, Cache-Controlo Response headers: Content-Type, Last-Modified, Set-Cookie, Cache-Control* What happens when you type a URL in a browser?o The browser “translate” the URL into an IP address via a DNSo Your browser sends a request to the corresponding server, including headers o The server sends a response to the cliento The browser creates the DOM from that HTML, including all related resources like e.g. css-files, js-files, images…
